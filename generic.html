FUN BUS

The fun bus site was created when I was learning about the way we users interact with the pages.This is known as EVENTS. Any time there is some interaction by way of a mouse, keyboard, etc., the DOM creates and propagates an event object. This event object carries information about the event so that it may be handled at any point up the tree from the point of origin.
