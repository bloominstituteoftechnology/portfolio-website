International Child Nutrition Status Tracking

Team Role: Front-End Framework Developer and Back-End Developer

The ICNST app tracks and organizes data for child malnutrition nonprofits around the globe. Users create country-specific accounts and can access and add data for children in their communities. The back-end was built in Node.js + Express using REST architecture. The front-end was built in ReactJS and uses Context API + React Hooks for application state management.

Built on a cross-functional team with one Web UI Developer and two Front-End Developers

Front-End Tech Stack: ReactJS, React Router, Context API, React Hooks, axios, recharts, styled-components

Back-End Tech Stack: Node.js, Express, bcryptjs, cors, dotenv, helmet, jsonwebtoken, knex, knex-cleaner, sqlite (development), nodemon (development), pg (production)
