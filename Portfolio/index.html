Recent Projects

Here are a few of the project I have be working on recently.

Student Tracker React App

This react app is built for professors to track their students progress. Professors can add students to their displayed list of students, add projects to those specific students that they are helping with, track messages sent to specific students and edit/delete project and student info. I collaborated with one other person to build the backend in Node.js with express and Postgress. The backend is deployed to Heroku. I built the front end using react and react router. I used Material-UI to style the UI. The front end is deployed on netlify.

Guidr

I worked as part of an eight person team to complete this project. I was responsible for the Home, About, and FAQ pages. I used my knowledge of HTML and CSS to format and style the pages. I also used Javascript to add various dynamic features such as event handlers, expandable and collapsable cards on the About page, and an accordion structure on the FAQ page.

Newspaper Website

This project demonstrates my knowledge of Javascript class constructors. I used class constructors and data attributes to allow the user to see the articles associated with the trending topics tabs as each tab is clicked.

User Cards

For this project I used axios to access the GitHub API to build cards based on data received from the API. This project demonstrates my knowledge of Javascript promises, array methods, selecting dom elements, and using functions to create HTML elements.

Nasa Photo of the Day

For this project I used react to build an app to access the nasa photo of the day api and used hooks to manage state, the user is able to use a dropdown menu to display photos from different dates.
