Projects and Experience

Miracle Messages: Front End Developer | Internship

09/2019 - 02/2020

I worked remotely alongside Miracle Messages and its CEO, using several streamline technologies to create a major upcoming application for the company. The application is privately hosted and is still under continual refinement as new Developers join the Miracle Messages team.

The Front-End of this application was built using languages such as JavaScript and React Redux, while also using such libraries as Material-UI, React-Map-GL, Sass, Formik, Yup and Axios. The Back-End side was built using Node.js with a SQL Database and libraries like Express.JS, Bcrypt.JS, Knex and Helmet.

My contribution to the Miracle Messages app was through the Front-End by creating and refining the UI with JavaScript and React Redux, while also managing component rendering with React Routes. I helped in optimizing the Styling by using Sass and Material-UI, and worked with the Back-End to establish fluid CRUD Operations and Axios calls.


Potluck Planner: Front End Developer

Potluck Planner was a project aimed to be an easy to use "event planning" app that allowed users to create, plan, connect and share Potluck Parties with their friends. This was a multi team project that required consistent communication between the teams and the ability to show strong leadership and resourcefulness to reach our goals.

The Front-End of this application was built using JavaScript and React Hooks, while also using such libraries as Bootstrap, Styled-Components, Semantic-UI, Context API, Yup, Formik and Axios. The Back-End side was built using Node.js with a SQL Database and libraries like Express.JS, Bcrypt.JS, Knex and Helmet.

I contributed to Potluck Planner through the Front-End by creating and refining the UI with JavaScript, React.js, JSX and Styled-Components while also managing component rendering with React Routes. I handled state management by using Context API and React Hooks, while also working in tandem with the Back-End using Axios, YUP, Formik and CRUD Operations.

Tips Ease: Back End Developer

Tips Ease is a tipping application for service workers to directly receive tips from customers or the public without 3rd party interventions.

The Back-End side was built using Node.js with a SQL Database and libraries such as Express.JS, BCrypt.JS, Nodemon, Jest.js, CORS, Cross-ENV, Super-Test, JSON-Web-Token, Knex and Helmet. The Front-End of this application was built using JavaScript and React.js along side React Hooks and React Routes.

I contributed to Tips Ease through the Back-End by creating the SQL Database and used Node.js along with other libraries such as Express.js, Knex and Nodemon to implement best practices and create efficient API Routes. I used middleware and technologies such as BCrypt.JS, JSON-Web-Token and CORS to help keep the Back-End Application secure.
