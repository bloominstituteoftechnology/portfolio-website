Miracle Messages | Front-End Developer

I worked remotely alongside Miracle Messages and its CEO, using several streamline technologies to create a major upcoming application for the company. I spearheaded the Front-End UI by building around the Mapbox application with React and Redux, while the Back-End was built from the ground up using Node.js and SQL. The application is privately hosted and is still under continual refinement as new Developers join the Miracle Messages team.

Potluck Planner | Front-End Developer

Potluck Planner was a project aimed to be an easy to use "event planning" app that allowed users to create, plan, connect and share Potluck Parties with their friends. The Back-End was built using Node.js and SQL while the Front-End side was built in JavaScript using the React Hooks and Redux frameworks while also using libraries such as Bootstrap, Semantic-UI, Styled-Components, Axios, Yup, Formik and JQuery. This was a multi team project that required consistent communication between the teams and the ability to show strong leadership and resourcefulness to reach our goals.

JavaScript Snake Game | Front-End Developer

Inspired by "The game Snake in 90 lines of JavaScript" article, I decided to undertake my own Snake game project by using JavaScript React to see how far I could expand upon it. The Front End is currently basic JavaScript React but I hope to further expand upon that by using Styled Components, Bootstrap, or other frameworks that could lend a more visual helping hand.

Python Adventure Game | Front-End Developer

This was a MUD adventure game project where the Back-End was built in Python using Graph like Data Structures, Django, Blockchain and GraphQL, and the Front-End was built using JavaScript React and Redux. The goal was to take the Graph like Data Structure and make it into a several hundred room game world that could easily be traversed by the user's profile/avatar, and have the ability to see other users in the same room.

Guidr App | Front-End Developer

The Guidr App was a project made for back country Guides that allowed them to advertise their own trips and journeys by updating, personalizing, and even privatizing their top trips and experiences. The Back-End side was built using Node.js and libraries like Bcrypt.JS, Express.JS, Knex and Helmet, while the Front-End was built using JavaScript React Hooks and also used such libraries as Semantic-UI, Less, Styled Components, Formik, Yup and Axios.

Guidr Responsive Landing Page | Front-End Developer

This project is a template for the Guidr App's landing page built almost solely with HTML and CSS. The landing page is meant to be a responsive first introduction to the Guidr App which was a project made for back country Guides that allowed them to advertise their own trips and journeys by updating, personalizing, and even privatizing their top trips and experiences. The Front-End was built using mainly HTML and CSS, some base JavaScript, and is very responsive.

NASA-photo-of-the-day | Front-End Developer

This JavaScript React App project uses an API from NASA to fetch the "Astronomy Photo Of The Day" or APOD, and then display it along with the corresponding information. The App was created using the git bash command "create-react-app" which builds a standard structured React App, then modified to use Axios calls and React Hooks for State management and logic.

TipsEase App | Back-End Developer

TipsEase is a JavaScript tipping app for service workers to directly receive tips from customers or the public without any 3rd party interventions. I hosted the Back-End on Heroku as well as structured the project using Node.js and a SQLite3 database. Other libraries I used were those such as Knex, Nodemon.js, Express.js, CORS, Jsonwebtoken and Bcrypt.js.
